# ** Extensor Wifi para ROM MIUI11 ** 

### ** Descripción ** 
Este módulo Magisk ayuda a habilitar la capacidad de habilitar WiFi y Hotspot en ROM MIUI11 en teléfonos Redmi.
Systemless aplique este módulo para evitar actualizaciones OTA. A partir de ahora se agregan Whyred y Violet. se agregarán más dispositivos de soporte después de la prueba.

##  Construir
###  Linux, BSD, macOS, Android
Requiere wget.
    wget -O META-INF / com / google / android / update-binary https://raw.githubusercontent.com/topjohnwu/Magisk/master/scripts/module_installer.sh
Y ZIP.
###  Otro
Descargue [ esto ] (https://raw.githubusercontent.com/topjohnwu/Magisk/master/scripts/module_installer.sh) y colóquelo en `META-INF / com / google / android / update-binary` . Y ZIP.
